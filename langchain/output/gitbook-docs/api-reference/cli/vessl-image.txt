# vessl image

### Overview

Run `vessl image --help` to view the list of commands, `vessl image [COMMAND] --help` to view individual command instructions.

### List all kernel images

```
vessl image list
```

### View information on the kernel image

```
vessl image read ID
```

| Argument | Description     |
| -------- | --------------- |
| `ID`     | Kernel image ID |
