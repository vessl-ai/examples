# Welcome to VESSL Docs!

{% embed url="https://www.youtube.com/watch?v=l_y5HCeOg6c" %}

****[**VESSL**](https://vessl.ai) is an end-to-end MLOps platform that allows researchers to train and monitor models in scale as a team. Machine learning systems are full of "[hidden technical debt](https://papers.nips.cc/paper/2015/file/86df7dcfd896fcaf2674f757a2463eba-Paper.pdf)". VESSL removes the complex and repetitive process of setting up pipelines and infrastructures so teams can focus on their models.&#x20;
