# Connecting Clusters to VESSL

To connect a cluster to VESSL, Click **New cluster** button on the cluster tab in your organization.

![](<../../../.gitbook/assets/Screen Shot 2022-02-25 at 1.09.26 PM.png>)

A dialog for connecting a new cluster will show up. You can connect two types of cluster to VESSL:

* [VESSL-managed cluster](vessl-managed-cluster.md)
* [User-managed cluster (custom cluster)](user-managed-cluster-custom-cluster.md)
