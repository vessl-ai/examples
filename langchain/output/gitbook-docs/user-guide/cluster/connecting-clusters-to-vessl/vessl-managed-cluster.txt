# VESSL-managed Cluster

VESSL provides cloud environment for the users who want to run ML workloads on the managed suite. When you create organization, VESSL will automatically connect your organization to the cluster located in the same region of organization. You can also connect other managed clusters provided by VESSL.

## Connect to a new VESSL-managed cluster

Select **"VESSL"** from the **"Type"** section on the new cluster dialog and click **"Next"** button.

![](<../../../.gitbook/assets/image (83).png>)

Select **region** and **cluster** from the dropdown and click **"Next"** button again - VESSL will register the selected cluster to the organization.&#x20;

![](<../../../.gitbook/assets/image (149).png>)
