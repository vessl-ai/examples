# Cluster Nodes

A **node** of a cluster is worker machine that handles a machine learning workload.

To see nodes information on the cluster, go to **"Nodes"** tab on the cluster management page.

![](<../../../.gitbook/assets/Screen Shot 2022-02-28 at 12.42.21 PM (1).png>)

Cluster nodes page will show up with the list of nodes connected to the cluster.

![](<../../../.gitbook/assets/image (151).png>)

To see details of the nodes, click the **checkbox** on the left of the row indicating each node. Node detail information includes:

* Node metadata: System and resource information of the node including GPU product type and capacity.
* System metrics: Real-time system metrics (resource usage and capacity) of the node.
* Workloads: Experiments and workspace containers) running on the node.
* Issues: Issues or problems reporting about a node's health.
